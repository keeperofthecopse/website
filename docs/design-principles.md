# Longspooning Archive Design Principles

## Core Principle

The site should feel like an archive, not a product.

## Tone

- Restrained
- Text-first
- Slightly institutional
- Not modern or stylised

## Layout

- Mobile-first
- Vertical flow
- No card-based UI
- No dashboard patterns

## Typography

- Serif for content
- Sans-serif for UI
- Strong hierarchy
- High readability

## Colour

- Off-white background
- Near-black text
- Green for links
- Brown for subtle structure only

## Containment

- Use background tone, not borders
- Avoid boxes and cards

## Interaction

- Minimal
- No animations
- No app-like behaviour

## Authority

- The Field > Archive > Eccles
- Design should reinforce this subtly

## Rule

If a design choice makes the site feel like a modern web product, reject it.
